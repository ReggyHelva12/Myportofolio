Perancangan Database Rumah Belajar Duta Semesta

Database Duta Semesta adalah proyek database yang dikembangkan sebagai fondasi Sistem Manajemen Pembelajaran berbasis web. Proyek ini berfokus pada perancangan skema database yang mampu mengintegrasikan berbagai proses akademik, mulai dari pengelolaan data siswa, guru, kelas, jadwal pembelajaran, absensi, penilaian tugas dan sikap, hingga penyusunan rapor dan hasil Sistem Pendukung Keputusan (SPK). Database dibangun menggunakan MySQL dan diimplementasikan melalui Laravel Migration, sehingga struktur database dapat dikelola secara terstruktur, mudah dikembangkan, serta mendukung performa aplikasi dalam mengolah data secara cepat, akurat, dan konsisten.
